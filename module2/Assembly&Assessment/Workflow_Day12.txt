## Steps
1. FastQC on raw reads → saved report as: YOUR_SAMPLEID_raw_fastqc.html
2. fastp trimming (default settings) → saved report as: YOUR_SAMPLEID_fastp_report.html
3. FastQC on trimmed reads → saved report as: YOUR_SAMPLEID_trimmed_fastqc.html
4. SPAdes assembly on trimmed reads → saved contigs as: YOUR_SAMPLEID_spades_contigs.fasta
5. QUAST evaluation on contigs → saved report as: YOUR_SAMPLEID_quast_report.pdf
